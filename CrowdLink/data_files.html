CrowdLink Data Files

This page consists of data files used as input, obtained as output, and analysis results of the experimental evaluation of CrowdLink: Crowdsourcing for Large-Scale Linked Data Management. All these files can be accessed [HERE].

In the following, it is explained in what those folders contain:

  • Input_and_Output_Files: This folder contains input and output files used during the experiments.
  • Input-Success-Result-Files: contains 3 types of files: “.input”, “.question”, and “.properties”. “.input” files have the input material for questions. “.question” files have the information for formats of the different questions. “.properties” files contain some information about the tasks such as reward amount, duration of availability of tasks etc. “.result” files contains outcome from the crowd as answers to the questions that we supplied.
    • Published Tasks: contains “.input”, “.properties” and “.question” files that belong to the published tasks.
    • Task Templates: contains the three of “.template” files that utilize the format of the question depending on the question type (acquire, review a link etc.)
  • Queries: contains SPARQL queries that are used for querying the LOD during the process of generating the questions.
  • Analysis_Files: contains analysis documentation, which include MS Excel “.xlsx” files and “.result” files. “.result” files are obtained as answers to the questions from the crowd. “.xlsx” files are more organized versions of those “.result” files, so we can evaluate the results in a more methodological way.
